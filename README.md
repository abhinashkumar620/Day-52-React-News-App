# News App

A **responsive news application** built with **React** and **Tailwind CSS** that fetches live news from [NewsAPI](https://newsapi.org/). Users can filter news by category, navigate pages, and read full articles.


## Features

- Fetches top headlines from the US using NewsAPI.
- Responsive layout:
  - Mobile: single column news cards.
  - Tablet/Desktop: grid layout (2-3 columns).
- Category selection:
  - General, Business, Entertainment, Health, Science, Sports, Technology.
- Pagination with previous/next buttons.
- Loading spinner while fetching data.
- Error handling if API fails.
- Modern card-based design with rounded corners, shadows, and hover effects.
- External links open in a new tab.
